### Setup

rails db:migrate RAILS_ENV=development 


### Clone this repository

```bash
git clone https://github.com/ogagaoghene/my-blog-app.git
```
### Move into the directory with

```bash
cd my-blog-app
```

### Run linter

```bash
rubocop .
```
